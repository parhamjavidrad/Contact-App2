# Contact App

یک پروژه ساده برای مدیریت لیست مخاطبین با React

---

## ویژگی‌ها

- افزودن، ویرایش و حذف مخاطب
- فرم افزودن/ویرایش با اعتبارسنجی yup و مدیریت react-hook-form
- استفاده از Context API برای مدیریت داده‌ها
- جستجو بین مخاطبین
